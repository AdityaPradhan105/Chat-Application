💬 Real-Time Chat Application
A full-stack, real-time chat application built using the MERN stack, designed to deliver seamless and instant communication between users. This project leverages WebSocket technology to ensure low-latency messaging and a responsive user experience.

✨ Features
⚡ Instant Messaging – Real-time communication using Socket.io

🔐 Authentication & Authorization – Secure login/signup with JWT

💾 Persistent Chat Storage – MongoDB for storing users and messages

🔔 Real-Time Notifications – Get notified instantly for new messages

🟢 Online/Offline Status – Track active users dynamically

📱 Responsive Design – Optimized for mobile, tablet, and desktop

🌐 RESTful APIs – Efficient backend with Express.js

🛠️ Tech Stack -
Frontend :

React.js

CSS / Tailwind

Backend:

Node.js

Express.js

Database:

MongoDB

Real-Time Communication:

Socket.io

Authentication:

JSON Web Tokens (JWT)

📌 Overview
This application demonstrates the implementation of real-time systems using WebSockets, combined with secure authentication and scalable backend architecture. It serves as a strong foundation for building modern chat platforms, collaborative tools, or messaging services.
